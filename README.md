## 소개

숭실대학교 IT지원위원회의 활동과 구성원을 소개하는 공식 홈페이지다.

## 개발 환경

```bash
pnpm install
pnpm dev
```

개발 서버는 [http://localhost:3000](http://localhost:3000)에서 확인할 수 있다.
